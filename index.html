Settings

When enabled, Export Map asks you to pick your custom_planets folder and saves the map directly into a matching subfolder there, instead of downloading a plain .png. Currently only works in Chromium-based browsers (Chrome, Edge). The folder is remembered after the first time, so you normally won't need to browse to it again - just a quick "allow access" confirmation.
More settings will be added here in the future.

Change Map Size

Adjust any edge independently: positive numbers add empty space, negative numbers crop. The center fields show (and can directly set) the resulting total size.
